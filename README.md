# DBMS-Notes

A well-organized collection of **Database Management System (DBMS)** notes for quick revision and learning.
This repository is helpful for **students, beginners, and anyone preparing for exams or interviews** related to DBMS.

---

## 📖 Topics Covered

* Introduction to Database
* Database Management System Overview
* DBMS Architecture
* Data Models
* Entity Relationship (ER) Model
* Relational Model
* Keys in DBMS
* Normalization
* Functional Dependencies
* Transactions and Concurrency Control
* Database Security
* SQL Basics

---

## 🎯 Purpose

This repository aims to:

* Provide **simple and easy-to-understand DBMS notes**
